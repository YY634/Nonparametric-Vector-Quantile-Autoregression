
title: Nonparametric Vector Quantile Autoregression

The folder "reproducibility_materials" contains:
1. The R codes to conduct simulations via implementing nonparametric VQAR, which replicate the simulation results.
2. The R codes for real data analysis.
3. Pre-processed real datasets that are ready to use. 


Before running the R scripts or knit the Worklow.Rmd file, please set folder "reproducibility_materials" as the working directory. 
Under the working directory  ("reproducibility_materials"), there should be two sub-directories--"simulations" and "real_data_analysis". 


The sub-directory "reproducibility_materials/simulations/" contains all the R scripts needed in Section 4.1 and Appendix B.1 of the paper. 
The procedure of simulations is described in Workflow.Rmd and Workflow_simulation.txt. 

The sub-directory "reproducibility_materials/real_data_analysis/" contains all the R scripts and dataset needed in Section 4.2 of the paper. 
The procedure of real data analysis is described in Workflow.Rmd and Workflow_realdata.txt.

The purpose of each R script has been explained in the script itself and in Workflow.Rmd and Workflow_realdata.txt.
